How to use these skills
1
Install the skills
Run bash install.sh in the repo root. Skills are copied into ~/.claude/skills/ and become available globally.
→
2
Open Claude
Use claude.ai in your browser, or the Claude Code CLI in your terminal. Skills work in both places.
→
3
Type a slash command
Every skill has a /skill-name command. Type it in the chat box and hit Enter to activate the skill.

Creative &
Planning

Skills that help you think clearly, generate ideas, and structure work before you build.

Section 01 — Creative & Planning
🧠
Brainstorming
Creative · Ideation
/brainstorming
Sometimes you don't need code — you need to think. The Brainstorming skill turns Claude into a structured creative partner. It helps you explore a problem from multiple angles, generate a wide range of options without judgment, and distill the best ideas into something actionable. Great for early-stage product decisions, naming, architecture trade-offs, or whenever you feel stuck and need fresh perspectives.
Example prompt"I want to brainstorm names for my new developer tool that helps with API testing"
Creative
📋
Planning
Strategy · Roadmap
/planning
The Planning skill turns fuzzy goals into concrete, ordered steps. Given a project or feature, it produces a structured plan with clear milestones, dependencies, and priorities — without jumping straight to code. Use it before any significant piece of work to get alignment on what needs to happen and in what order. It keeps things high-level enough to be useful for stakeholders, yet specific enough to actually execute against.
Example prompt"Plan how to add Stripe payments to my SaaS app — I need subscriptions and one-time purchases"
Strategy

Developer
Skills

Core engineering skills — reviewing, fixing, testing, and refining code.

Section 02 — Developer Skills
🔍
Code Review
Quality · Security · Performance
/code-review
Gives your code the same treatment a senior engineer would. It checks correctness, flags security issues, spots performance bottlenecks, and calls out style inconsistencies — with clear, constructive feedback rather than nitpicks. Useful before merging any PR or after finishing a feature.
Example prompt"Review the changes I just made to the auth module — focus on security and edge cases"
Quality
🐛
Debugging
Bug fixing · Root cause analysis
/debugging
Systematic bug hunting. Rather than guessing, this skill reads error messages, traces execution paths, and narrows down root causes methodically. It explains what's wrong, why it's wrong, and fixes it — without touching unrelated code.
Example prompt"Getting TypeError: cannot read property 'map' of undefined on line 42 — here's the stack trace…"
Fixing
✅
TDD · Test-Driven Development
Testing · Red-green-refactor
/tdd
Guides you through the red-green-refactor cycle properly. Write a failing test, implement just enough code to make it pass, then clean up. This skill keeps you honest about requirements and produces naturally well-tested code from the start.
Example prompt"I want to build a price calculator function using TDD — start with the tests"
Testing
✨
Simplify
Refactor · Quality pass
/simplify
Runs a quality pass over code you just wrote — looking for redundant logic, missed reuse opportunities, and unnecessary complexity. It doesn't refactor for fun; it fixes what actually needs fixing and leaves the rest alone.
Example prompt"I just finished the checkout flow — simplify it, there's probably duplicated logic"
Refactor
🌿
Git Worktrees
Parallel branches · Multi-tasking
/git-worktrees
Lets you work on multiple branches at the same time without stashing or constantly switching context. Each worktree is an independent checkout — perfect for reviewing a colleague's PR while still building your own feature.
Example prompt"Set up a worktree so I can review Sarah's PR while I keep working on my feature branch"
Git
⌨️
Keybindings
Shortcuts · Claude Code config
/keybindings
Customise your Claude Code keyboard shortcuts. Add chord bindings, change the submit key, or remap any action — all saved to ~/.claude/keybindings.json. Makes Claude Code feel exactly how you want it.
Example prompt"Change the submit key from Enter to Ctrl+Enter, and add Ctrl+K to clear the chat"
Config
Section 02 — Developer Skills (cont.)
🔁
Loop
Recurring tasks · Polling · Automation
/loop
Run any prompt or slash command on a recurring schedule. Useful for polling a deployment status, running a health check every few minutes, or automating any repeated task without babysitting the terminal. Just specify the interval and the command — Loop handles the rest. Not for one-off tasks; only invoke when repetition is the actual goal.
Example prompt"/loop 5m /gsd-progress" — checks and reports your GSD project status every 5 minutes
Automation

GSD

Get Stuff Done — a structured system for planning, building, and shipping features from start to finish.

Section 03 — GSD Workflow
🚀
GSD · New Project
Bootstrap · Scaffold · Setup
/gsd-new-project
The starting point for any new software project. This skill scaffolds the folder structure, installs tooling, configures environments, and gets you to a clean first commit — so you can spend energy on the actual problem rather than boilerplate setup.
Example prompt"Start a new Next.js SaaS app with Tailwind, Prisma, and Stripe — TypeScript, monorepo structure"
Bootstrap
🗺️
GSD · Plan Phase
Implementation plan · Before coding
/gsd-plan-phase
Before writing a single line of code, this skill breaks a feature or task into a concrete, ordered implementation plan. It forces you to think through edge cases and dependencies upfront — reducing rewrites and surprises during execution.
Example prompt"Plan how to add real-time notifications to our app using WebSockets"
Planning
⚙️
GSD · Execute Phase
Build · Implement · Ship
/gsd-execute-phase
The doing phase. Given a plan, this skill works through each step methodically — writing code, running tests, and keeping track of what's done. It avoids scope creep and stays focused on shipping what was planned.
Example prompt"Execute the WebSocket notifications plan we just made — start with the backend"
Build
🔎
GSD · Debug
Focused bug fix · Within GSD workflow
/gsd-debug
A focused debugging session within a larger GSD workflow. When execution hits a bug, this skill isolates it, identifies the root cause, and resolves it — then hands control back to the execute phase without losing momentum.
Example prompt"The WebSocket connection keeps dropping after 30 seconds — debug this before we continue"
Debug
📊
GSD · Progress
Status update · What's done · What's next
/gsd-progress
A status report for your current GSD workflow. It shows what's been completed, what's in progress, and what's still to come — keeping you and your team aligned without digging through commit logs.
Example prompt"Where are we on the notifications feature? What's left to do?"
Status

Section 04 — Figma & Design
🎨
Figma · Implement Design
Design → Code · Figma URL or screenshot
/figma-implement-design
Share a Figma link or design screenshot and this skill converts it into production-ready frontend code — whether that's HTML/CSS, React, or another framework. It respects spacing, typography, colours, and component structure, producing clean code rather than rough approximations.
Example prompt"Implement this Figma frame as a React component using Tailwind: figma.com/file/…"
Design → Code
📐
Figma · Design System Rules
Design tokens · Component guidelines
/figma-design-system-rules
Takes your Figma design system and extracts formal rules — design tokens, component usage guidelines, spacing conventions, and colour semantics. The output keeps your codebase and designs in sync, and gives engineers a clear reference for how to use the system correctly.
Example prompt"Extract the design rules from our Figma system and create a rules file for the repo"
Design System
🔗
Figma · Code Connect
Dev Mode · Component linking
/figma-code-connect
Sets up Figma Code Connect so that when designers inspect a component in Figma Dev Mode, they see the actual code snippet from your codebase — not a generated approximation. Closes the loop between your component library and your design file, making handoff effortless.
Example prompt"Set up Code Connect for our Button and Modal components in the Figma design file"
Dev Mode

Read, search, create, and query your Notion workspace — all from Claude.

Section 06 — Notion Integration
🔍
Notion · Find
Locate page by title
/notion-find
Locates a specific Notion page by title or partial title match and returns its ID and URL. The fast way to jump to a page when you know roughly what it's called but don't want to click around.
Example prompt"Find my 'Q1 Roadmap' page in Notion"
Notion
🔎
Notion · Search
Full workspace search
/notion-search
Searches across your entire Notion workspace — pages and databases — by keyword. Great for discovery when you're not sure exactly where something lives, or to explore what exists on a given topic.
Example prompt"Search Notion for anything related to our pricing strategy"
Notion
📄
Notion · Create Page
New page · Notes · Wiki
/notion-create-page
Creates a new Notion page with content you specify. Whether it's a meeting note, a wiki entry, or a design doc — just tell Claude what to write and where, and it's done without opening the browser.
Example prompt"Create a Notion page titled 'Sprint 12 Retro' under the Engineering section with today's notes"
Notion
✅
Notion · Create Task
Add database record · Task · To-do
/notion-create-task
Adds a new record to any Notion database — a task, a project, a CRM entry, whatever your database represents. Supports setting properties like status, priority, date, and assignee without touching the Notion UI.
Example prompt"Add a task 'Fix login bug' to our Tasks database, priority High, due Friday"
Notion
🗄️
Notion · Database Query
Filter · Sort · List database records
/notion-database-query
Query any Notion database with filters, sorting, and pagination. Pull a list of all open tasks, items due this week, entries with a specific tag — whatever slice of data you need. The result comes back cleanly so you can act on it immediately, without building a Notion integration yourself.
Example prompt"List all tasks in my Tasks database where status is 'In Progress', sorted by due date"
Notion

Section 07 — Social & Advertising
📸
Instagram · Fetch
Download post · Reel · Extract metadata
/instagram-fetch
Paste an Instagram post, reel, or story URL and this skill downloads the media, extracts captions, and pulls metadata — all using yt-dlp under the hood. Works for public content out of the box; private content requires Instagram credentials. Useful for archiving content, analysing competitors, or repurposing media.
Example prompt"Fetch this Instagram reel and give me the caption and all metadata: instagram.com/reel/…"
Social Media
💡
Instagram · Prompt Extractor
Reverse-engineer AI images · Creative research
/instagram-prompt-extractor
Reverse-engineers the creative prompt or concept behind an AI-generated Instagram image. Share a post and this skill analyses the visual style, composition, and caption to deduce the most likely prompt — so you can recreate or riff on it. Great for creative research and understanding trending AI aesthetics.
Example prompt"What AI prompt was likely used to create this image? [paste Instagram post URL or image]"
AI Creative
📣
Adspirer
Google Ads · Meta Ads · Campaign management
/adspirer
Manage Google Ads and Meta (Facebook/Instagram) advertising campaigns directly from Claude. Create campaigns, analyse performance metrics, write ad copy, optimise budgets, and troubleshoot underperforming ads — without leaving the terminal. Built for marketers and founders who want to move fast without switching between ad dashboards all day.
Example prompt"Analyse my Meta ad campaigns for this week and tell me which ones to pause and which to scale"
Advertising
