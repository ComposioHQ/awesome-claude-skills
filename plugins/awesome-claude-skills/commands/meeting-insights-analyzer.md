# Meeting Insights Analyzer

Load the `awesome-claude-skills:meeting-insights-analyzer` skill and follow its workflow to analyze a meeting.

If `$ARGUMENTS` is provided, use it as the path to the transcript or a description of the meeting.

Otherwise, ask the user to provide a meeting transcript or recording to analyze.
