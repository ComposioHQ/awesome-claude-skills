# Tailored Resume Generator

Load the `awesome-claude-skills:tailored-resume-generator` skill and follow its workflow to generate a resume.

If `$ARGUMENTS` is provided, use it as the job description or role to tailor the resume for.

Otherwise, ask the user to provide the job description and their existing resume or experience.
