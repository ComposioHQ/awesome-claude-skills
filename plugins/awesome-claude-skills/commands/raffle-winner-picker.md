# Raffle Winner Picker

Load the `awesome-claude-skills:raffle-winner-picker` skill and follow its workflow to pick winners.

If `$ARGUMENTS` is provided, use it as the participant list or file to pick winners from.

Otherwise, ask the user to provide their list of participants and how many winners to select.
