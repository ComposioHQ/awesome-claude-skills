# Domain Name Brainstormer

Load the `awesome-claude-skills:domain-name-brainstormer` skill and follow its workflow to generate domain names.

If `$ARGUMENTS` is provided, use it as the project description or keywords to brainstorm around.

Otherwise, ask the user to describe their project or the keywords they want in a domain name.
