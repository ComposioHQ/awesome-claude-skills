# MCP Builder

Load the `awesome-claude-skills:mcp-builder` skill and follow its workflow to build an MCP server.

If `$ARGUMENTS` is provided, use it as the name or API for the MCP server to build.

Otherwise, ask the user what external service or API they want to integrate via MCP.
