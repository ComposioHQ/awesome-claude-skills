# Invoice Organizer

Load the `awesome-claude-skills:invoice-organizer` skill and follow its workflow to organize invoices.

If `$ARGUMENTS` is provided, use it as the directory containing invoices and receipts to organize.

Otherwise, ask the user where their invoices are stored and what tax period to organize for.
