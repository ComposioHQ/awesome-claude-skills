# Skill Share

Load the `awesome-claude-skills:skill-share` skill and follow its workflow to create and share a skill.

If `$ARGUMENTS` is provided, use it as the skill name or description to create and share.

Otherwise, ask the user what skill they want to create and which Slack channel to share it in.
