# Slack GIF Creator

Load the `awesome-claude-skills:slack-gif-creator` skill and follow its workflow to create a Slack GIF.

If `$ARGUMENTS` is provided, use it as the description of the animated GIF to create.

Otherwise, ask the user what they want to animate and for which Slack context.
