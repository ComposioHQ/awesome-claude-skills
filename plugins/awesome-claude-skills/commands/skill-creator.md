# Skill Creator

Load the `awesome-claude-skills:skill-creator` skill and follow its workflow to create a new skill.

If `$ARGUMENTS` is provided, use it as the name or description of the skill to create.

Otherwise, ask the user what skill they want to build and what problem it should solve.
