# Connect Apps

Load the `awesome-claude-skills:connect-apps` skill and follow its workflow to connect to external apps.

If `$ARGUMENTS` is provided, use it as the target app or action to perform.

Otherwise, ask the user which external service they want to connect to and what they want to do.
