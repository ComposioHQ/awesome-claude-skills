# Video Downloader

Load the `awesome-claude-skills:video-downloader` skill and follow its workflow to download a video.

If `$ARGUMENTS` is provided, use it as the YouTube URL or video to download.

Otherwise, ask the user for the video URL, desired quality, and output format.
