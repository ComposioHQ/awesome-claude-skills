# Artifacts Builder

Load the `awesome-claude-skills:artifacts-builder` skill and follow its workflow to build a frontend artifact.

If `$ARGUMENTS` is provided, use it as the description of what to build.

Otherwise, ask the user what kind of artifact they want to create.
