## Notion:database-query
**Category:** Notion Integration

**What it does:**
Queries a Notion database by name or ID and returns structured, readable results. Use to search or filter database content.

**When to trigger:**
- "Query my Notion [database name]"
- "Show me all tasks in Notion where status is X"

**How to install:**
```bash
npx claude install superpowers
```
Also requires the **Notion MCP server** connected.

**Trigger phrase:** Ask Claude to query or search a Notion database.
