## Notion:create-task
**Category:** Notion Integration

**What it does:**
Creates a new task in your Notion tasks database with sensible defaults. Quick way to log a task without leaving Claude Code.

**When to trigger:**
- "Add a task to Notion"
- "Create a Notion task for X"

**How to install:**
```bash
npx claude install superpowers
```
Also requires the **Notion MCP server** to be connected to Claude Code.

**Trigger phrase:** Ask Claude to create a task in Notion.
