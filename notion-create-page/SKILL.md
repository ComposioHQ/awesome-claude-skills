## Notion:create-page
**Category:** Notion Integration

**What it does:**
Creates a new Notion page, optionally under a specific parent, using the Notion Workspace Skill and Notion MCP server.

**When to trigger:**
- "Create a Notion page about X"
- "Make a new page under [parent]"

**How to install:**
```bash
npx claude install superpowers
```
Also requires the **Notion MCP server** connected.

**Trigger phrase:** Ask Claude to create a Notion page.
