## instagram-prompt-extractor
**Category:** Utility / Creative

**What it does:**
Reverse-engineers AI generation prompts from any image or video. Works with URLs from Instagram, TikTok, Twitter/X, Pinterest, Reddit, Midjourney, DALL-E, Civitai, ArtStation, and direct image links — or local files.

**Supported content:** Portraits, landscapes, architecture, fashion, food, abstract art, video clips — anything AI-generated.

**When to trigger:**
- "What prompt made this?"
- "Extract the prompt from this image"
- "How was this generated?"
- "Reverse engineer this"
- "What AI made this?"
- "Recreate this style"
- User pastes any URL with an image/video and asks about AI generation

**How to install:**
```bash
npx claude install superpowers
```

**Trigger phrase:** Share any image/video URL and ask how to recreate it or what AI prompt made it.
