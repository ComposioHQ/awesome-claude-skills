## Notion:search
**Category:** Notion Integration

**What it does:**
Searches your entire Notion workspace using the Notion MCP server. Performs semantic search across pages, databases, and connected sources.

**When to trigger:**
- "Search Notion for X"
- "Find anything in Notion about X"

**How to install:**
```bash
npx claude install superpowers
```
Also requires the **Notion MCP server** connected.

**Trigger phrase:** Ask Claude to search Notion for any topic.
