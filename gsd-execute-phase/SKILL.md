## gsd:execute-phase
**Category:** GSD (Get Stuff Done)

**What it does:**
Executes all plans in a phase using wave-based parallelization. Spawns multiple subagents to work on independent tasks simultaneously, then merges results. The core execution engine of GSD.

**When to trigger:**
After a phase is planned and ready to execute.

**How to install GSD:**
```bash
npx claude install superpowers
```

**Trigger phrase:** `/gsd:execute-phase`
