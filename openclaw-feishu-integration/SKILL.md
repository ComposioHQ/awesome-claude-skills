# OpenClaw 飞书集成 Skill

让 OpenClaw 能够操作飞书的各种功能，包括消息、文档、日历和任务。

## 何时使用此 Skill

- 需要发送消息到飞书群聊或私聊
- 需要读取或创建飞书云文档
- 需要管理飞书日历和会议
- 需要创建或查询飞书任务
- 需要搜索飞书知识库

## 此 Skill 做什么

### 能力 1: 飞书消息管理
- 发送文本、图片、文件消息
- 回复消息和话题
- 搜索历史消息
- 下载消息中的图片和文件

### 能力 2: 飞书文档操作
- 创建云文档和 Wiki
- 读取和更新文档内容
- 插入图片和文件
- 管理文档评论

### 能力 3: 日历和会议
- 创建和管理日程
- 查询忙闲状态
- 邀请参会人
- 创建视频会议

### 能力 4: 任务管理
- 创建、更新、删除任务
- 管理任务清单
- 添加评论
- 创建子任务

### 能力 5: 多维表格
- 创建和管理 bitable
- 操作记录（增删改查）
- 管理字段和视图

## 使用示例

**基础用法:**
```
帮我给飞书群 "研发团队" 发送一条消息：今晚 8 点开会
```

**进阶用法:**
```
创建一个飞书文档，标题是 "项目周报"，内容包括本周完成的工作
```

**示例:**
- 用户说："在飞书日历上创建一个明天下午 3 点的会议"
- 输出：调用 feishu_calendar_event 创建日程

**灵感来自:** OpenClaw 飞书插件的日常工作流

## 常见用例

- 发送日报/周报到飞书群
- 自动创建会议邀请
- 从飞书文档提取信息
- 管理飞书任务清单
- 查询团队成员忙闲状态
