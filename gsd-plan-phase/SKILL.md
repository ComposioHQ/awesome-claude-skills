## gsd:plan-phase
**Category:** GSD (Get Stuff Done)

**What it does:**
Creates a detailed phase plan (`PLAN.md`) with a task breakdown, dependency analysis, and a verification loop. Runs a research agent, a planning agent, and a plan-checker agent to ensure quality before execution.

**When to trigger:**
Before executing a phase.

**How to install GSD:**
```bash
npx claude install superpowers
```

**Trigger phrase:** `/gsd:plan-phase`
