# OpenClaw 浏览器自动化 Skill

让 OpenClaw 能够自动化浏览器操作，包括网页浏览、表单填写、数据抓取等。

## 何时使用此 Skill

- 需要自动化网页操作
- 需要抓取网页数据
- 需要填写表单
- 需要截图或录屏

## 此 Skill 做什么

### 能力 1: 网页导航
- 打开网页
- 点击链接和按钮
- 填写表单
- 滚动页面

### 能力 2: 数据抓取
- 提取网页内容
- 获取表格数据
- 下载图片/文件

### 能力 3: 表单操作
- 输入文本
- 选择下拉菜单
- 勾选复选框
- 提交表单

### 能力 4: 截图和录屏
- 截图当前页面
- 录制操作视频

### 能力 5: 等待和验证
- 等待元素加载
- 验证页面状态

## 使用示例

**基础用法:**
```
打开百度并搜索 OpenClaw
```

**进阶用法:**
```
填写这个表单并提交
```

**示例:**
- 用户说："截图当前页面"
- 输出：执行浏览器截图

**灵感来自:** OpenClaw 浏览器自动化日常工作流

## 常见用例

- 自动化网页测试
- 数据采集和抓取
- 表单批量提交
- 网页监控和告警
