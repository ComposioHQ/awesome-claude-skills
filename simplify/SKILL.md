## simplify
**Category:** Utility

**What it does:**
Reviews recently changed code for opportunities to reuse existing utilities, improve quality, and increase efficiency. After reviewing, it fixes any issues found. Think of it as a post-implementation polish pass.

**When to trigger:**
Invoke after writing or editing code:
- `/simplify`

**How to install:**
Bundled with Superpowers:
```bash
npx claude install superpowers
```

**Trigger phrase:** `/simplify` — run after completing a feature or fix to clean up the implementation.
