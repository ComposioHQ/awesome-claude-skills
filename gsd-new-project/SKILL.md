## gsd:new-project
**Category:** GSD (Get Stuff Done)

**What it does:**
Initializes a brand new project with deep context gathering and a `PROJECT.md` file. Runs parallel research agents, creates a full roadmap, and sets up your `.planning/` directory.

**When to trigger:**
At the very start of a new project.

**How to install GSD:**
```bash
npx claude install superpowers
```

**Trigger phrase:** `/gsd:new-project`
