## gsd:progress
**Category:** GSD (Get Stuff Done)

**What it does:**
Checks overall project progress, shows current context (active milestone, phases, completion status), and routes you to the next action (execute or plan). Your project dashboard command.

**How to install GSD:**
```bash
npx claude install superpowers
```

**Trigger phrase:** `/gsd:progress`
