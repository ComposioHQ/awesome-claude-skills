# Newsletter Curator

Streamlines the newsletter creation process by collecting content from multiple sources, organizing it by themes or topics, generating concise summaries, and suggesting optimal layouts. Helps content creators maintain consistent publishing schedules while ensuring high-quality, engaging newsletters.

## When to Use This Skill

- Compiling weekly or monthly newsletters from multiple sources
- Summarizing articles, blog posts, or news for subscribers
- Organizing content by themes or categories
- Creating editorial calendars and content pipelines
- Repurposing existing content into newsletter format

## What This Skill Does

1. **Content Aggregation**: Collects and processes articles, links, and sources
2. **Thematic Organization**: Groups content by topics, trends, or categories
3. **Summary Generation**: Creates concise, engaging summaries of each piece
4. **Layout Suggestions**: Recommends structure and visual hierarchy
5. **Headline Creation**: Suggests catchy subject lines and section headers

## How to Use

### Basic Usage

```
Create a newsletter from these 5 articles about AI development
```

### Advanced Usage

```
Curate a weekly developer newsletter from these sources: organize by topic, write summaries, suggest subject lines, and recommend a layout with featured and secondary sections
```

## Tips

- Provide 3-5x more content than your final newsletter length for curation options
- Specify target audience (technical level, interests) for better tone
- Include previous newsletters for consistency in style and format
- Request multiple subject line options to A/B test

## Common Use Cases

- Weekly team/company internal newsletters
- Developer community digests
- Industry trend roundups
- Content marketing newsletters
- Educational course updates
