## figma:implement-design
**Category:** Figma

**What it does:**
Translates Figma designs into production-ready code with 1:1 visual fidelity. Handles layout, spacing, colors, typography, and component structure.

**When to trigger:**
- Implementing UI from Figma files
- "Implement this design"
- "Generate code from this Figma"
- User provides a Figma URL
- "Build this component matching Figma specs"

**How to install:**
```bash
npx claude install figma
```
Also requires the **Figma MCP server** connected to Claude Code.

**Trigger phrase:** Share a Figma URL and ask to implement or build the design.
